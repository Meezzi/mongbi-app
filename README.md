# 1. Project Overview (프로젝트 개요)
- 프로젝트 이름: 몽비
- 프로젝트 설명: 몽비는 꿈을 먹고 해석해주는 해몽 도깨비 꿈의 비밀을 풀어준다는 의미를 가지고 있죠
사용자의 입력한 꿈을 바탕으로 내면의 감정과 심리를 이해하고 정서적 치유와 자기 성장을 돕는 AI기반 꿈 해석 및 심리 케어 서비스입니다.

# 3. Key Features (주요 기능)
- **소셜 로그인**:
  - ANDROID : 카카오 , 네이버 소셜로그인 제공
  - IOS: 애플, 카카오 , 네이버 소셜 로그인 제공

- **꿈 입력 및 AI 꿈 해석**:
  - 사용자가 꾼 꿈을 작성 후 감정을 선택 할 수 있습니다.
  - 작성한 꿈과 선택한 감정을 기반으로 AI가 꿈에 기반한 심리 상태 , AI의 해설 그리고 AI의 한마디를 제공해줍니다.

- **꿈 기록**:
  - 캘린더에 내가 작성한 꿈 감정을 표시해줍니다.
  - 작성한 꿈 , AI 해석 내용을 작성한 일자별로 표시합니다.

- **꿈 통계**:
  - 월간 / 연간을 기준으로 사용자가 입력한 꿈을 기반으로 기분분포, 키워드 분포 등 여러가지 통계를 시각화 해줍니다.

# 7. Development Workflow (개발 워크플로우)
## 브랜치 전략 (Branch Strategy)
우리의 브랜치 전략은 Git Flow를 기반으로 하며, 다음과 같은 브랜치를 사용합니다.

- Main Branch
  - 배포 가능한 상태의 코드를 유지합니다.
  - 모든 배포는 이 브랜치에서 이루어집니다.
  
- {name} Branch
  - 팀원 각자의 개발 브랜치입니다.
  - 모든 기능 개발은 이 브랜치에서 이루어집니다.

# 9. 커밋 컨벤션
## 기본 구조
```
type: subject

body 
```

<br/>

## type 종류
```
feat : 새로운 기능 추가
bug : 버그 수정
refactor  : 리팩토링 
```

<br/>

## 커밋 예시
```
== ex1
Feat: "회원 가입 기능 구현"
SMS, 이메일 중복확인 API 개발

== ex2
add: font 파일 추가

== ex3
bugfix: 상품 등록이 안되는 오류 수정

== ex4
refactor: 불필요한 위젯 제거 및 정리
```
